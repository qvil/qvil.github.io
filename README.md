## Prerequisites

- Node.js >= 12.x (Recommended the latest version)
- Gatsby - [Install the Gatsby CLI
  ](https://www.gatsbyjs.com/docs/quick-start/#install-the-gatsby-cli)
- (Yarn(Recommended))

## Install

```sh
npm install -g gatsby-cli

yarn
# or
npm install
```

## Development

```sh
gatsby develop
```

## Deploy

```sh
yarn deploy
```
