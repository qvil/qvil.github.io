Next.js 9.5부터 추가된 Headers 기능을 사용하면 원하는 Path에 HTTP 헤더를 설정할 수 있다.

> 예를 들어 `/fonts` 디렉토리의 폰트를 캐싱하고 싶은 경우.

```tsx
module.exports = {
  async headers() {
    return [
      {
        source: "/fonts/:font*",
        headers: [
          {
            key: "Cache-Control",
            value: "public, max-age=31536000",
          },
        ],
      },
    ];
  },
};
```
