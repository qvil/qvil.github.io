## URL query param이 있는 경우에만 API 호출 시 query param을 전달하는 경우 많이 사용한다.

```tsx
const { query: { someQuery } } = context;
const { data } = await fetchSomething({
  params: {
    ...(someQuery && { someQuery }),
  },
});
```
