## 객체는 기본적으로 값이 아니라 참조를 비교하기 때문에 값이 같아도 참조가 다르면 다른 객체라고 본다.

```js
const obj1 = { a: 1 };
const obj2 = { a: 1 };

console.log(obj1 === obj2); // false
```

## 이를 해결하기 위하여 객체를 문자열로 치환하여 비교하는 방법이 있다.

```js
const obj1 = { a: 1 };
const obj2 = { a: 1 };

JSON.stringify(obj1) === JSON.stringify(obj2); // true
```
