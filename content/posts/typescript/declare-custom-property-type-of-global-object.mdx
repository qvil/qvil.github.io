## Global object(Browser: window/Node.js: global)에 custom property를 추가해야되는데 타입에러가 생기는 경우

아래와 같이 타입 선언을 해서 사용

```ts
declare global {
  interface Window {
    myCutstomProperty: string;
  }
  namespace NodeJS {
    interface Global {
      myCutstomProperty: string;
    }
  }
}
```
