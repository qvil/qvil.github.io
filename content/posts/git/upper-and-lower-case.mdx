## Git에서 파일, 폴더 대소문자 변경 시 변경사항을 인식 못하는 경우

아래와 같이 config를 바꿔준다.

```sh
git config core.ignorecase false
```

Replace된 변경 사항 인식을 못한 경우 아래와 같이 해당 파일, 폴더를 Remove --cached 해준다.

```sh
git rm -r --cached dir_name
```
